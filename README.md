# Grossify Mini Grocery

A simple grocery shopping website built with React.

## Features

- Product listing
- Product search
- Product details
- Add to cart
- Remove from cart
- API integration
- State management using Zustand

## Tech Stack

React  
Zustand  
Axios  
React Router  

## API Used

https://fakestoreapi.com

## Deployment

Vercel / Netlify
